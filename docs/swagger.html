Como autenticar

Todos os endpoints (exceto GET /v1/health) exigem autenticação via header: Authorization: Bearer <sua-api-key>

As API Keys têm o formato mh_tenant-slug_<hex64> e são geradas pelo administrador.

1

Receba sua API Key do administrador do Media Hub

2

Clique em Authorize 🔒 abaixo e cole a key no campo ApiKey

3

Teste com GET /v1/health/auth para confirmar que está funcionando

4

Crie um upload e envie seus arquivos de mídia
